📋 UX Principles Applied

  • Clarity First: Every error message explains what happened in plain language
  • Status Always Visible: Booking status is the most prominent element
  • Recovery Focused: Every error screen provides 2+ actionable next steps
  • Reassurance: Calm tone and guidance reduce user anxiety
  • Support Access: Help options are always accessible
  • Preserved Identity: IRCTC branding and color scheme maintained throughout
  • Responsive Design: Works seamlessly on desktop, tablet, and mobile
  • Accessibility: Clear hierarchy, good contrast, semantic HTML
