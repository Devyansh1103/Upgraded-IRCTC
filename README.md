# IRCTC UX Redesign

This project is a **UX/UI redesign concept for the IRCTC website**, focused on improving usability, clarity, and booking efficiency while **strictly preserving the existing IRCTC theme, branding, and official trust-based identity**.

The redesign addresses key user pain points such as a **cluttered homepage, stressful booking flow, overloaded search results, dense passenger forms, payment confusion, and weak error handling**. Instead of rebranding the platform, this project improves the **information hierarchy, layout structure, booking journey, accessibility, responsiveness, and overall user experience**.

The redesigned solution includes improvements for:
- Homepage hierarchy
- Train search results
- Passenger details flow
- Payment experience
- Error and loading states
- Accessibility and mobile responsiveness

The goal is to make IRCTC more **intuitive, efficient, and user-friendly** without losing its familiar visual identity.
